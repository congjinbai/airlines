更新版本说明

准备新版本：

  1. 将最新的 index.html 文件保存到桌面上
  2. 插入外接移动硬盘
  3. 打开终端，将下方的命令复制粘贴到终端内，按回车执行
1 一键更新命令
bash -c 'cd "/Volumes/Lexar/航班信息生成器源文件/flight-info-generator" && cp ~/Desktop/index.html ./ && npm run build && open dist/'
2 在打开的 dist 文件夹中，找到新的"航班信息生成器-1.0.0.dmg"文件，双击安装
⚠️ 重要提示：

1. 请确保桌面上有最新的 index.html 文件

2. 请确保移动硬盘已经连接，并且"更新航班信息生成器.sh"脚本文件和"flight-info-generator"文件夹在"航班信息生成器源文件"夹里

3. 更新前会自动备份当前版本，备份文件为 index.html.backup.时间戳

4. 更新完成后需要手动安装新的 DMG 文件

5. 所有已保存的数据（设置、历史记录、付款状态）将保持不变
